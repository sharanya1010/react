What is useMemo?

useMemo is a React Hook that memorizes the result of a calculation. Instead of recalculating on every render, React reuses the cached value until its dependencies change.

Why do we need it?

Every time a component re-renders, all the code inside it runs again. If you have an expensive calculation, this can slow down your app.

useMemo solves this by running the calculation only when required.

Syntax

const memoizedValue = useMemo(() => { return calculation; }, [dependencies]);

The value is cached until one of the dependencies changes.
